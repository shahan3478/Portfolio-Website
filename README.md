🚀 Shahan's Developer Portfolio

Welcome to my personal portfolio website! This project is a modern, responsive, and visually appealing showcase of my journey as a Full Stack Web Developer. It highlights my technical skills, work experience, and the diverse range of projects I have built.

✨ Features

Dynamic Typing Animation: Powered by Typed.js to display rotating roles (Full Stack Developer, Web Designer).

Modern Dark UI: A sleek Navy Blue and Purple aesthetic designed for readability and a professional feel.

Work Experience Timeline: A unique vertical layout showcasing a progression from Frontend clones to Full Stack MERN applications.

Fully Responsive: Optimized for various screen sizes using CSS Flexbox.

Interactive Elements: Hover effects on navigation and buttons for an enhanced user experience.

🛠️ Tech Stack

Frontend: HTML5, CSS3 (Flexbox, Google Fonts), JavaScript (ES6).

Libraries: Typed.js for the hero section animation.

Design: Custom CSS with a focus on modern typography (Poppins) and dark-mode aesthetics.

📂 Project Highlights (Featured in Experience)

My portfolio lists several key milestones in my development career:

Frontend Replicas: High-fidelity clones of Netflix, Amazon, and Starbucks.

React News App: A dynamic application fetching real-time data from external APIs.

URL Shortener: A backend-focused project using Node.js and Express.js.

Weather App: Real-time weather updates via API integration.

MERN Note App: A full-stack CRUD application (MongoDB, Express, React, Node).
